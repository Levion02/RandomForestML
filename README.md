Модель машинного обучения для прогнозирования киберугроз

Прогнозирование возникновения киберугроз с использованием Random Forest
Данный проект посвящён созданию модели машинного обучения для прогнозирования вероятности возникновения киберугроз для компаний.
Проект был выполнен в рамках дипломной работы бакалавриата. В ходе работы были проанализированы реальные статистические данные о киберинцидентах за 2024 год
, представленные в отчёте Verizon Data Breach Investigations Report (DBIR). На основе аналитических данных из отчёта был сформирован структурированный датасет
, отражающий ключевые характеристики и факторы, связанные с возникновением киберугроз.
Полученный датасет был использован для обучения и тестирования модели машинного обучения Random Forest
, которая позволяет оценивать вероятность возникновения угроз на основе различных параметров и характеристик.

Основные этапы проекта:
Анализ статистики киберугроз на основе отчёта Verizon DBIR
Формирование и подготовка датасета на основе выявленных факторов
Предобработка данных и подготовка признаков для обучения модели
Обучение модели Random Forest с использованием библиотеки Scikit-learn
Разделение данных на обучающую и тестовую выборки
Оценка качества модели и тестирование на новых ("живых") данных

Используемые технологии
Python
pandas
NumPy
scikit-learn

Цель проекта — продемонстрировать применение методов машинного обучения для анализа и прогнозирования киберугроз, 
а также получить практический опыт работы с подготовкой данных, обучением моделей и анализом их эффективности.
